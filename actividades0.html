Actividad Nro 1.- Diseñemos un circuito mixto de resistencias en serie y paralelo con una batería, siguiendo estos pasos básicos en la plataforma de Fritzing

Circuito mixto de resistencias en serie y paralelo.

¿Qué materiales crees que vamos a necesitar?

  • Componentes:
  • 5 resistencias con valores: R1=5 Ω, R2=8 Ω, R3=10 Ω, R4=12 Ω, R5=4 Ω
    1 batería de 9V
  • Protoboard y cables para conexiones

Paso N°1: 

Abre Fritzing y selecciona la vista de Protoboard.

Paso N°2:

Arrastra las 5 resistencias y la batería desde la biblioteca de componentes.

Paso N°3:

Abre Fritzing y selecciona la vista de Protoboard.

Paso N°4:

Conecta las resistencias en el protoboard formando una combinación de serie y paralelo según el diseño que quieras practicar.

Paso N°5:

Conecta la batería para alimentar el circuito.

Paso N°6:

Cambia a la vista Esquemático para observar el diagrama generado automáticamente y verificar las conexiones.

Código con Errores

La imagen muestra un ejemplo de código (sketch) programado en el entorno Arduino IDE, que tiene como objetivo encender un LED cuando se presiona un botón. El programa define dos pines: uno para el LED (pin 13) y otro para el botón (pin 7). Luego, en la función setup(), se establece qué tipo de función tendrá cada pin (entrada o salida), y en la función loop() se evalúa el estado del botón para decidir si se enciende o apaga el LED.

Sin embargo, este código contiene errores comunes que son útiles para el aprendizaje inicial:

El LED está configurado como entrada en lugar de salida.
El botón está configurado como salida en lugar de entrada.
Se está usando un solo signo igual (=) en una condición, cuando debería ser doble (==) para comparar valores.
También se recomienda utilizar el nombre de la constante BOTON en lugar del número 7 directamente, para mantener el código más legible y ordenado.

Instrucciones

Objetivo de la actividad: Desarrollar una secuencia de leds  funcional mediante el uso de Arduino IDE y Fritzing.

Los estudiantes, se organizan en grupos pequeños de (3 a 4 integrantes), donde diseñan en el programa Fritzing un circuito básico compuesto por tres LEDs conectados a la placa Arduino Nano. Cada LED debe encenderse en una secuencia específica.
Una vez diseñado el circuito, los estudiantes programan dicha secuencia utilizando Arduino IDE, y comprobar si la programación ejecuta correctamente el comportamiento esperado.

Materiales.

Computadoras con Fritzing y Arduino IDE instalados.
Placa Arduino Nano.

Código base (Arduino IDE)

void setup() {

  pinMode(2, OUTPUT);

  pinMode(3, OUTPUT);

  pinMode(4, OUTPUT);

}

void loop() {

  digitalWrite(2, HIGH);

  delay(500);

  digitalWrite(2, LOW);

  digitalWrite(3, HIGH);

  delay(500);

  digitalWrite(3, LOW);

  digitalWrite(4, HIGH);

  delay(500);

  digitalWrite(4, LOW);

}
